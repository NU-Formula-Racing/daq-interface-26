# Design reference

HTML/CSS/JS prototypes for the dashboard UI. These are visual references, not
production code — recreate them in the React app under `app/src/`.

## Contents

- `project/` — prototype HTML, CSS, components, and assets. `project/FSAE Dashboard.html` is the primary design.

## Usage

Read the HTML and CSS directly to pull dimensions, colors, and layout rules.
The React implementation should match the visual output without copying the
prototype's internal structure.
